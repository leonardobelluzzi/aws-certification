Gateway de Arquivos do Amazon S3: o Gateway de Arquivos do Amazon S3 comporta uma interface de arquivos no Amazon Simple Storage Service (Amazon S3) e combina um serviço e um dispositivo de software virtual. Quando você usa essa combinação, é possível armazenar e recuperar objetos no Amazon S3 por meio de protocolos de arquivo padrão do setor, como o Network File System (NFS) e o Server Message Block (SMB).
