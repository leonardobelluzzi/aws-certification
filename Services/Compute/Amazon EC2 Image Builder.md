
O EC2 Image Builder é um serviço totalmente gerenciado que simplifica a personalização, o teste, a distribuição e o gerenciamento do ciclo de vida de imagens de máquina da Amazon (AMI) e imagens de contêineres.

Manter as AMI e as imagens de contêiner atualizadas pode demandar muito tempo, consumir vários recursos e causar propensão a erros. Na prática, os clientes atualizam e capturam snapshots manualmente de instâncias do EC2 ou investem no desenvolvimento de scripts de automação personalizados para manutenção de imagens.


## USE CASE

A company is looking at a service/tool to automate and minimize the time spent on keeping the server images up-to-date. These server images are used by Amazon Elastic Compute Cloud (Amazon EC2) instances as well as the on-premises systems.

Which AWS service will help achieve the company's need?

Amazon EC2 Image Builder
