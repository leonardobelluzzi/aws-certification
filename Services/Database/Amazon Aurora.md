
Amazon Aurora is a MySQL and PostgreSQL-compatible relational database built for the cloud. Amazon Aurora is fully managed by [Amazon Relational Database Service (RDS)](RDS.md), which automates time-consuming administration tasks like hardware provisioning, database setup, patching, and backups.


## USE CASE

Which of the following entities applies patches to the underlying OS for Amazon Aurora?

The AWS Product Team automatically


## USE CASE

A company is planning to move their traditional CRM application running on MySQL to an AWS database service. Which database service is the right fit for this requirement?

Amazon Aurora
