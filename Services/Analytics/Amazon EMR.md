
Amazon EMR, which was previously called Amazon Elastic MapReduce, is a managed cluster platform that simplifies running big data frameworks, such as Apache Hadoop and Apache Spark, on AWS to process and analyze vast amounts of data. Using these frameworks and related open source projects, you can process data for analytics purposes and business intelligence workloads. Amazon EMR also lets you transform and move large amounts of data into and out of other AWS data stores and databases, such as [Amazon Simple Storage Service (Amazon S3)](Technology/Storage/S3.md) and [Amazon DynamoDB](Technology/Database/NoSQL/Amazon%20DynamoDB.md)

### Apache Spark

O **Apache Spark** é uma plataforma de processamento de dados em larga escala, projetada para ser **rápida, distribuída e versátil**. Ele é usado principalmente em ambientes de **Big Data** para realizar análises complexas com alta performance.

### Hadoop

O **Hadoop** é uma plataforma de software de código aberto desenvolvida pela Apache que permite o **armazenamento e processamento distribuído de grandes volumes de dados** — estamos falando de gigabytes a petabytes! Ele é especialmente útil em ambientes de **Big Data**, onde a escalabilidade, tolerância a falhas e eficiência são essenciais.
